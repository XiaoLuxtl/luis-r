# 🚀 AI-Driven Portfolio & Automation Infrastructure

Bienvenido a mi portafolio personal y entorno de pruebas de ingeniería. Este proyecto no es solo una web estática; es el frontend de una infraestructura diseñada para demostrar capacidades en **IA, Automatización (n8n) y Administración de Servidores**.

📍 **Live Demo:** [https://xiaoluxtl.github.io/luis-r/](https://xiaoluxtl.github.io/luis-r/)

---

## 🛠️ Stack Tecnológico

### Frontend & UI
• **Astro:** Framework de alto rendimiento con arquitectura de islas y SSG (Static Site Generation).
• **Tailwind CSS:** Diseño responsivo y optimizado.
• **Typebot:** Interfaz conversacional para la interacción con el Agente de IA.

### Backend & Infrastructure (Self-Hosted)
• **n8n:** Orquestador de flujos de trabajo alojado en entorno local/VPS.
• **Docker:** Contenerización de servicios para asegurar portabilidad y escalabilidad.
• **Nginx:** Configurado como Reverse Proxy para la gestión segura de peticiones.
• **OpenAI (GPT-4o mini):** Motor de procesamiento de lenguaje natural integrado mediante API.

---

## 🧩 Características Destacadas

• **Agente de IA Personalizado:** El sitio integra un chatbot capaz de responder consultas técnicas sobre mi trayectoria, proyectos y stack tecnológico en tiempo real.
• **Arquitectura Híbrida:** Despliegue estático en GitHub Pages que se comunica mediante Webhooks con mi infraestructura privada de automatización.
• **SEO & Performance:** Optimización técnica para tiempos de carga mínimos y alta indexabilidad.
